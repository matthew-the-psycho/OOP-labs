Требования:
Реализуйте описанное в презентации приложение и добавьте в него:
    - ещё один способ заполнения матрицы (1);
    - ещё одну вычисляемую характеристику (2).

С точки зрения графики, интерфейс выглядит чуть иначе, чем в презентации :: расположение элементов.

*(1) :: Реализован способ заполнения треугольной матрицы;
*(2) :: Реализована характеристика : Сумма максимумов from each line.

Техническая особенность:
Итоговый запуск происходил из файла Main2.java,
    Который является сборкой всех описанных классов в одном месте,
    для работоспособности программы.

Код первого запуска contain'ится в Main.java 
    и неизменённом (по выше описанной причине) MatrixData.java,
