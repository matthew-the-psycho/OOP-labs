Требование:
Создайте любое приложение, 
    где есть обработка 
    хотя бы одного исключения.

Один из вариантов:
    Создать массив (список);
    пользователь вводит индекс; 
    вывести значение элемента с этим индексом; 
    если индекс за пределами массива, 
        то выбрасывается исключение

Использован шаблонный Класс MeinVector, 
    код которого был написан ещё в 2022 году (5 семестр :: Теория Информации, Кодирования и Передачи Данных)
//Для обработки исключений необходимо было добавить структуру.

Для реализации данного задания графика -- необязательна.
Следовательно, ./Lab5.elf is cli-based application.
